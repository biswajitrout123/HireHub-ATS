# 🚀 SmartJobPortal - MERN Stack Job Board

SmartJobPortal is a full-stack application designed to connect recruiters with job seekers. It features a modern UI, secure authentication, and a powerful search engine for job listings.

## ✨ Key Features
- **Role-Based Access Control (RBAC):** Distinct dashboards for Recruiters and Job Seekers.
- **Advanced Job Filtering:** Search by keyword, location (including Remote), and salary.
- **Recruiter Tools:** Create, manage, and delete job postings. View applicants for each job.
- **Job Seeker Profiles:** Customizable bios, skill tags, and resume management.
- **Modern UI:** Built with Tailwind CSS, featuring Skeleton Loaders for a premium feel.
- **Security:** Protected routes using JWT (JSON Web Tokens) and bcrypt password hashing.

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS, Lucide React (Icons), Axios.
- **Backend:** Node.js, Express.js.
- **Database:** MongoDB (Mongoose).
- **State Management:** React Hooks (useState, useEffect).
